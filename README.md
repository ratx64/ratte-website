# RatteCS Link Tree

A central hub for RatteCS social media profiles, affiliate links, and discount codes. Built with React, TypeScript, and Tailwind CSS, optimized for SEO and affiliate revenue.

## Features

- 🎮 Centralized social media and affiliate links
- 💰 Optimized for affiliate revenue and coupon sites
- 🔍 SEO-friendly with rich snippets and structured data
- 🎯 AEO optimized for voice search and featured snippets
- 👤 EEAT signals (Person schema, expertise indicators)
- ♿ WCAG 2.1 AA accessible
- 🔒 Security headers and CSP
- 🌓 Dark creator-native theme
- 📱 Fully responsive design
- 🚀 Blazing fast performance with Cloudflare Pages
- 🤖 Web scraper and search engine friendly

## Tech Stack

- React + TypeScript
- Tailwind CSS
- rspack/rsbuild
- Cloudflare Pages

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is configured for deployment on Cloudflare Pages. Follow these steps:

1. Push your code to GitHub
2. Connect your repository to Cloudflare Pages
3. Configure the following settings:
   - Build command: `npm run build`
   - Build output directory: `dist`
   - Node version: 18
4. Deploy!

## SEO & Optimization

The site is optimized for:

- Search engines (Technical SEO, On-Page SEO)
- Answer Engine Optimization (AEO) - Voice search and featured snippets
- EEAT (Experience, Expertise, Authoritativeness, Trustworthiness)
- Coupon sites and affiliate aggregators
- Web scrapers and crawlers

### Key SEO Features

- ✅ Comprehensive meta tags (title, description, Open Graph, Twitter Cards)
- ✅ Structured data (JSON-LD) for WebSite, Person, and VideoObject
- ✅ XML sitemap with image entries
- ✅ robots.txt configuration
- ✅ Canonical URLs
- ✅ Affiliate link optimization (rel="sponsored")
- ✅ Mobile-friendly and responsive design
- ✅ Fast performance (lazy loading, optimized images)

### Accessibility (WCAG 2.1 AA)

- ✅ Skip-to-content link for keyboard navigation
- ✅ Semantic HTML with proper ARIA labels
- ✅ Screen reader compatible
- ✅ Keyboard navigation support
- ✅ Proper color contrast ratios
- ✅ Descriptive alt text for images

### Security

- ✅ Content Security Policy (CSP)
- ✅ Security headers (HSTS, X-Frame-Options, etc.)
- ✅ HTTPS enforcement
- ✅ Security.txt for responsible disclosure

### Performance

- ✅ Lazy loading for images and embeds
- ✅ Optimized Core Web Vitals
- ✅ Fast page load times
- ✅ Efficient asset delivery via Cloudflare CDN
